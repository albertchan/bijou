Bijou (Frontend)
================

The '/client' directory contains the frontend code for the Bijou web application skeleton.

## Frontend structure

```
client/
├── gulp/
├── scripts/
│   ├── actions/
│   ├── components/
│   ├── containers/
│   ├── reducers/
│   ├── stores/
│   └── app.js
├── scripts/
├── gulpfile.js
└── package.json
```
